When working with API calls, we will be returned a JSON Object. As we can see from the JSON Object below, they can be quite difficult to determe which values go with each attribute.

This example isn't too bad, but if we are returned an Object that has a lot more attributes, then we will struggle to determine what goes where. An example of this can be found HERE. This JSON Object shows all the possible mounts in the game World of Warcraft. There must be a better way to view these JSON Objets. Well, my Apprentance Engineer, you're in luck! It's time to fix that broken "I win" button and make our lives a lot easier!

The tool we will be using is called JSONView. This allows adds formatting to our Object and makes it much easier for us to work with. Just like we learned in our earlier programming classes, code is a lot more readable and maintainable when it is formatted in a specific manner. I found out about this Chrome extension from The Coding Train. This YouTube series has a lot of wonderful tutorials in a wide array of topics. I highly recommend his materials.

How to Download JSON

This only works in Google Chrome. If you use another browser search for their version of this addon.

  1. Google "Chrome Extension JSONview"
  2. Click on the link
  3. Click the Green button to add
  4. KABLAM! You're done!

Now, with this new extension your JSON Object will now look like this!

As you can see, this is a ton easier to read and you can now quickly determine what values go with each attribute. We will be using this exact JSON Object later on and we will go into more detail then about how to use it.


You should now have your own API Key and should have JSONView set up. It's now time to get started with your first API call. Click the next arrow and we will get started!
